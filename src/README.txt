How to start application?

Open "app.R" under the project folder in RStudio and click on "Run App" at the top right hand of the Source panel.


Note:
- Do make sure that you have an internet connection.
- The application may take some time to boot up as it is retrieving data from the internet. This may vary based on your internet connection.
- It is recommended to use the application in fullscreen.
- Should there be any packages that you have not installed, you can run "Package_Installation.R" to install them.
